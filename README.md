# C++-rpg
I made a game in c++ to get used to c++ syntax its just a text based story game but i learned more than i expected i had never tought it was this hard to make a ton of possibilites!

# Installation
  Just download the exe-file <br>
  run it <br>
  If it doesnt work try installing msys2<br>
# how to play
   launch the exe file <br>
   You will be asked questions that determine the story <br>
   if you mis-typed something you gotta restart the game
# Compileing source code
   download the source code<br>
   install g++
   compile it using the code runner extension in vscode (run in terminal) <br>
